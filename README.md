# GST Billing Backend

Spring Boot backend application for GST billing and invoice management.

## Features
- User authentication (JWT)
- GST billing APIs
- Invoice management
- Email notifications
- Database integration (PostgreSQL)

## Tech Stack
- Java
- Spring Boot
- Spring Security
- JWT
- PostgreSQL
- Maven

## How to Run
1. Clone the repository
2. Configure database in application-local.yml
3. Run the Spring Boot application

## Note
Sensitive configuration files are excluded using .gitignore.
Copy application-example.yml to application.yml and update local credentials.
